# DevFlix
App de streaming de vídeo estilo Netflix.



:dart:  **Justificativa**

App desenvolvido com proposito de integrar API do Youtube em um app Expo e Native.


:white_check_mark:   **Tecnologias**

- React Native
- JavaScript

:white_check_mark:   **Freameworks:**

- Expo
- npm
- Axios
